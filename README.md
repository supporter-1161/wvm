# WireGuard VPN Manager (WVM)

WVM — это bash-скрипт для автоматизированной настройки и управления WireGuard VPN-сервером на VPS с целью безопасного удаленного доступа к домашней сети в режимах Split и Full Tunnel.

## Возможности

*   Установка и настройка WireGuard VPN-сервера на VPS.
*   Поддержка двух режимов: Split Tunnel (доступ только к домашней сети) и Full Tunnel (весь трафик через VPS + доступ к домашней сети).
*   Управление клиентами: добавление, удаление, генерация конфигов.
*   Резервное копирование и восстановление конфигурации.
*   Мониторинг состояния VPN-туннеля.
*   Миграция на новый VPS с сохранением ключей и IP-адресов.

## Требования

*   VPS с Ubuntu 20.04/22.04 (или другая Debian-совместимая ОС).
*   Root-права на VPS.
*   Постоянное подключение к VPS от домашнего шлюза (роутера), настроенного как WireGuard-клиент.

## Установка и использование

1.  Склонируйте репозиторий:
    ```bash
    git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
    cd <ПАПКА_РЕПОЗИТОРИЯ>
    ```
2.  Запустите скрипт от имени root:
    ```bash
    sudo ./wg-setup.sh
    ```
3.  Следуйте инструкциям в меню.

## Структура

*   `wg-setup.sh`: Главный исполняемый скрипт.
*   `lib/`: Модульные bash-скрипты с функциями.
*   `templates/`: Шаблоны для генерации клиентских конфигов.

## Важно

*   Скрипт создает и управляет файлами в `/etc/wireguard/`.
*   Убедитесь, что у вас есть доступ к домашнему шлюзу и он готов к настройке WireGuard-соединения с VPS.
